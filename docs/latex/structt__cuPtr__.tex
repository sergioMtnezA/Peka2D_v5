\doxysubsection{Detailed Description}
Arrange the pointers to CUDA memory for all the run-\/control and data-\/arrays in \mbox{\hyperlink{structs_8h_a35d362804d71cc7cb2c7f57fe532a938}{t\+\_\+arrays}}. 

The definition of the variables and memory size is the same as that in the \mbox{\hyperlink{structs_8h_a35d362804d71cc7cb2c7f57fe532a938}{t\+\_\+arrays}} description. This structure is created in the RAM memory but the CUDA memory for data-\/arrays is allocated using these pointers.
